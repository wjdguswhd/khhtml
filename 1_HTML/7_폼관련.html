폼 관련 태그

form태그는 html에서 사용자가 입력한 값을 서버(자바)로 넘기는 역할을 함
form태그 내에 input태그에 작성한 데이터 -> 서버 - action 속성 : 폼에 입력된 데이터들을 전송할 주소 입력
- method 속성 : 전송 방식 지정(get : 쿼리스트링 노출 / post : 쿼리스트링 숨겨짐)

submit버튼을 눌렀을 때 method에 지정한 방식으로 action에 지정한 경로에 input태그의 값 전달
