스크립트 언어

기본 프로그램의 동작을 사용자의 요구에 맞게 수행되도록 해주는 용도로 사용
매우 빠르게 배우고 짧은 소스코드 파일로 상호작용 하도록 고안되었음

자바스크립트

웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체지향 프로그래밍 언어

자바스크립트 사용법

1. 태그에 직접 간단한 소스코드를 작성해서 실행하는 방법
2. script태그에 작성하여 실행하는 방법
3. 별도의 js파일로 작성해서 가져다 사용하는 방법

단, 소스의 함수 단위로 작성하고 이벤트 속성(on 이벤트명)을 사용하여 구동
이벤트 속성은 태그마다 차이가 있음

태그에 직접 자바스크립트 코드를 작성하여 실행

script태그 안에 함수 단위로 소스 작성하여 실행

별도의 js파일로 작성해 실행

자바스크립트의 데이터 입출력

데이터 출력

document.write()로 출력

window.alert()로 알림창 출력

console.log()로 콘솔화면에 출력

개발자도구 콘솔 화면에 출력하고자 할 때 , 디버깅할 때 주로 사용

innerHTML로 요소 내용 변경

자바스크립트에서 태그 앨리먼트의 값을 읽거나 값을 변경할 때 innerHTML속성 사용

데이터 입력

window.confirm()으로 입력
